# 🏋️ Tino Jr — Landing Page

> Landing page profissional para nutricionista esportivo e personal trainer, desenvolvida como projeto freelance e publicada em produção.

**[→ Ver site ao vivo](https://tino-jr-nutricionista-e-personal-tr.vercel.app)**

---

## Sobre o projeto

Desenvolvida para **Tino Jr** (CRN 5102 · CREF 020907-G/BA), a página concentra todos os planos da Metodologia TJR em uma única URL de alta conversão. O público principal é feminino e acessa pelo Instagram via celular — o que direcionou todas as decisões de layout, hierarquia e UX.

O projeto saiu de um briefing completo (identidade visual, textos, planos e preços) até a integração real com a plataforma de pagamento **Prime Coaching**, com mais de 20 links de checkout mapeados e atribuídos aos botões corretos.

## Funcionalidades

- Navbar fixa com backdrop-filter e borda ao rolar
- Hero com headline de impacto e dois CTAs
- Barra de estatísticas (500+ alunos · 8 anos · 95% aderência · 4.9/5)
- Seção "Sobre" com bio, credenciais profissionais (CRN/CREF) e cards
- Seção de serviços com cards e destaque no mais procurado
- Metodologia em 3 etapas numeradas
- **Planos com 4 abas dinâmicas:**
  - Treino Online — Sprint / Pro / Advanced / Black Edition
  - Híbrido Treino + Dieta — Hybrid Protocol / Hyper-Focus / Elite System
  - Nutrição Online e Presencial — tabela comparativa com links individuais
  - Personal Presencial + Dieta — One on One / Duo Start / Sprint Combined
- Depoimentos com cards de 5 estrelas
- Formulário de contato com pills de objetivo e envio direto via WhatsApp (mensagem gerada automaticamente)
- Animações de entrada com `IntersectionObserver`
- Menu hamburguer para mobile com animação CSS
- 100% responsivo — mobile-first
